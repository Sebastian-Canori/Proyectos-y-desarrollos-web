MI FORMA DE TRABAJO

Proceso de proyecto web

REUNION

Llevaremos a cabo una reunión para conocer mejor tu negocio o marca y así luego definiremos el contenido y las herramientas que vallamos a utilizar para llevar a cabo el proyecto y poder solventar todas las dudas.

PLANIFICACION

La segunda etapa consta de conciliar los tiempos de trabajo, esto significa que se establecerá un acuerdo entre ambas partes. De esta manera ordenaremos las etapas del proyecto en el tiempo estipulado.

LANZAMIENTO

Ejecutaremos el proyecto respetando todo lo estipulado anteriormente. Vale aclarar que a veces por cuestiones de falta de material del cliente ( texto e imágenes ). El proceso se alargue más de lo estipulado.
